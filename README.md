# About The App
The ML Wallet is a smart and convenient stored-value card system that allows drawing and sending cash and paying bills with just a few taps on your smartphones.
Features like:
* Send Money
* Receive Money
* Withdraw Money
* Bank Transfer
* Bills Payment
* E-Loading
* Viewing Transaction History
* etc
